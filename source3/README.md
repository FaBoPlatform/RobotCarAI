##### Lidar起動
3個搭載するLidar Lite v3はPower on時にすべて同一のアドレスとなるため、個々を識別するために1個ずつ配線し、アドレス変更を行う必要があります。
Power offになると変更したアドレスを失うため、Power on時に毎回行う必要があります。
